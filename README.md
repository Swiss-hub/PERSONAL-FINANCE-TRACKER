# PERSONAL FINANCE TRACKER WITH MACHINE LEARNING INSIGHTS

**Description:**  
A web-based personal finance tracker with Django backend and React frontend, allowing users to record income/expenses, view dashboards, and receive basic machine learning insights for smarter financial decisions. Designed for academic demonstration and full-stack learning.

---

## Features

- **User Authentication:** Secure registration and login
- **Transaction Management:** Add, edit, and delete income or expenses
- **Dashboard & Analytics:** Visual summaries with charts
- **Machine Learning Insights:** Predictive recommendations
- **Responsive Design:** Accessible on multiple devices

---

## Technology Stack

- **Backend:** Python, Django, Django REST Framework, PostgreSQL/SQLite
- **Frontend:** React (JavaScript + SWC), React Router v6
- **Machine Learning:** scikit-learn, pandas, NumPy
- **Others:** Git, Node.js, npm
